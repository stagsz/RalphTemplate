# Story 2.2: Contact list view (table, search, filters)

**Epic:** Epic 2 - Contact Management | **Story ID:** 2.2 | **Status:** Todo
**Estimated Time:** 2 days

## Description
Display all contacts in a searchable, filterable table view.

## Acceptance Criteria
- [ ] Table with columns: Name, Email, Company, Status, Actions
- [ ] Full-text search across name, email, company
- [ ] Filter by status (Lead, Customer)
- [ ] Pagination (25 per page)
- [ ] Sort by name, created date
- [ ] Loading states
- [ ] Empty state for no contacts
- [ ] Mobile responsive table

## Dependencies
**Depends On:** Story 2.1

## Definition of Done
All acceptance criteria met, unit and E2E tests written, code reviewed and merged
